SDK Quick Start
1
Install the SDK
pip install agentshield
2
Register an agent above, then initialize the client
from agentshield import AgentShieldClient

shield = AgentShieldClient(
    base_url="http://localhost:8000",
    agent_id="<your-agent-id>",
    api_key="<your-api-key>",
)
3
Intercept before each tool call
result = await shield.intercept(
    tool="database",
    action="query",
    prompt=user_request,           # optional: full user prompt for PII scanning
    tool_input={"query": sql},     # optional: structured input for policy matching
)

# shield raises PermissionError automatically if blocked
# or check manually:
if not result.allowed:
    return f"Blocked: {result.message}"
4
Use the @shield decorator (optional)
from agentshield import shield as shield_decorator

@shield_decorator(tool="filesystem", action="write")
async def write_file(path: str, content: str):
    # AgentShield intercepts automatically before this runs
    with open(path, "w") as f:
        f.write(content)
5
Direct HTTP (no SDK required)
curl -X POST http://localhost:8000/proxy/intercept \
  -H "X-API-Key: <your-api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "<your-agent-id>",
    "tool": "database",
    "action": "delete",
    "tool_input": {"table": "users"}
  }'
